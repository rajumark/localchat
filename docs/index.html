LocalChat

An on-device LLM chatbot powered by Gemini Nano via Android AICore. No API calls. No internet. 100% private.

🤖 Gemini Nano 🔒 On-Device 📱 Android 14+ 🎨 Material 3 🆓 Free
⬇ Download APK View on GitHub

Features

Everything you need from a modern on-device AI assistant.

🔒

100% Private

All inference runs locally on your device. Zero data ever leaves your phone. No servers, no tracking.

⚡

No Internet Required

Works completely offline once the model is downloaded. Perfect for travel, remote areas, or privacy.

🧠

Gemini Nano

Powered by Google's on-device foundation model via Android AICore. Runs on NPU or CPU.

🎨

Material Design 3

Modern UI with dynamic color (Material You). Adapts to your system theme — light or dark.

⬇️

Auto-Download

Checks model availability on launch and downloads Gemini Nano with a single tap.

🆓

Free & Open Source

No subscriptions, no API costs, no hidden fees. MIT licensed — use it any way you like.
