LocalChat

On-device LLM chatbot powered by Gemini Nano via Android AICore. No API calls. No internet. 100% private.

Gemini Nano On-Device Android 14+ Material 3
Download APK View on GitHub

Features

100% Private

All inference runs locally on your device. Zero data ever leaves your phone.

No Internet Required

Works completely offline once the model is downloaded. No servers or tracking.

Gemini Nano

Powered by Google's on-device foundation model via Android AICore. Runs on NPU or CPU.

Material Design 3

Modern UI with dynamic color (Material You). Light and dark mode support.

Auto-Download

Checks model availability on launch and downloads Gemini Nano with a single tap.

Open Source

No subscriptions, no API costs. MIT licensed — free forever.
